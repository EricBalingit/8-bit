Quick Setup

Define canvas with optional "resolution" attributes:

<canvas width="950" height="720" data-width="320" data-height="240"></canvas>

Get a 8-bit context:

var ctx = canvas.getContext("8-bit");

Now it can be used as a regular 2D context.

Options

Options can be passed to define palette and size:

var ctx = canvas.getContext("8-bit", {width:320, height:240, palette: _8bit.Palettes.CBM64});

If specified, width and height will override those specified in the canvas tag.

A palette should be defined, otherwise it will default to Commodore 64 palette (CBM64). You can also set palette later using the property of same name:

ctx.palette = _8bit.Palettes.AMIGA;

Images

See _8bit.Image for loading and dithering an image (you can still use a regular Image object though).

Colors

To use index for stroke and fill you need to turn off strict mode:

ctx.strict = false;
ctx.strokeStyle = 7;   // index 7 in current palette

You can otherwise use regular color schemes and they will be converted to an index internally representing the closest color.
